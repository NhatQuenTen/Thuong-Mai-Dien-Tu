Import Users từ Firebase Auth

Do hạn chế bảo mật, không thể import users hiện có từ client-side.

Các giải pháp:

  • 1. Sử dụng Firebase Admin SDK trên server để tạo API import
  • 2. Thêm users vào Firestore manually qua Firebase Console
  • 3. Đăng ký lại users để tự động lưu vào Firestore
